# Nova Digital Library (NDL) 
**Engineered by AICON Systems**

An enterprise-grade, Online-to-Offline (O2O) book rental platform featuring a Dual-Tier logistics checkout, hybrid physical-digital inventory bridging, and secure Role-Based Access Control (RBAC).

## Features
- **Dynamic QR Routing:** Bridges physical library shelves to digital express checkouts.
- **Multi-Item Cart System:** Built via session-state memory for batch processing.
- **RBAC Security:** Invisible administration dashboard locked behind staff authentication.
- **Financial Integration:** Direct Selar gateway mapping for standard and elite logistics tracking.

## Stack
- **Frontend:** Python (Streamlit), Custom CSS
- **Backend/Storage:** Supabase (PostgreSQL), REST APIs
